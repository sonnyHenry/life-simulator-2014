# 《2014:我的十二年》人生模拟器 — 游戏设计文档 v1.0

> 2026-07-03 与 Claude 对话确定。灵感与引擎参考:vc-simulator.com(已逆向其结局判定系统)。

## 一、核心定位

- **一句话**:从 2014 年高考开始,模拟一个中国年轻人到 2026 年的十二年人生。
- **基调**:苦涩幽默 + 温情(vc-simulator 同款气质),现实主义扎心事实配自嘲式幽默,关键节点给温情,追求"金句截图传播"。
- **形态**:React + Vite 单页应用,纯写死内容,零运行成本,可静态部署。
- **重玩性来源**:随机家境抽卡 × 高考分数浮动 × 4 条职业线 × 事件池随机抽取 × 多结局。

## 二、开局系统(2014 年 6 月)

流程:**家境抽卡 → 选省份 → 选文理 → 高考答题 → 志愿填报**

### 1. 随机家境抽卡(重玩性核心)
| 家境 | 初始金钱 | 特殊影响 |
|---|---|---|
| 城市中产 | 高 | 2016 年父母可资助买房首付 |
| 县城体制内 | 中 | 考公线加成,父母催回老家事件 |
| 农村家庭 | 低 | 学费压力事件,"小镇做题家"专属结局线 |
| 拆迁暴发户 | 很高 | 初始 10 万,更容易参与早期买房/投资,但不再直接碾压剧情 |
| 个体户家庭 | 中 | 家里生意受电商冲击的支线 |

### 2. 省份(简化为 4 档原型)
高考大省(河南/山东,分数线+30)/ 直辖市(北京/上海,分数线-40)/ 普通省份(基准)/ 西部省份(分数线-20)。

### 3. 高考答题
6~8 道 2014 年风格简单题(语文病句、数学函数、英语完形各 1~2 题 + 文综/理综 2 题),答对率映射分数段(如 6/8 → 一本线上 30~60 分,带随机浮动)。

### 4. 志愿填报
- 档次:985 冲一冲 / 211 稳一稳 / 一本保底 / 二本 / 复读(重考一次,时间成本 1 年)
- "冲"有滑档概率;专业选择决定职业线。

## 三、数值系统(0–100,金钱除外)

| 属性 | 初始值 | 作用 | 对标 vc-simulator |
|---|---|---|---|
| 学识 | 由高考分数决定 | 考试/求职/晋升成功率 | trackRecord |
| 金钱 | 由家境决定(单位:元) | 投资本金、买房、结局判定 | aum |
| 心态 | 70 | **≤0 触发提前结局**,低于阈值锁温情事件 | health |
| 人脉 | 10 | 解锁内推/合伙/相亲等隐藏选项 | network |
| 健康 | 85 | 长期透支型选择的隐性代价,过低触发提前结局 | - |

隐藏变量:家境标签、NPC 好感度、已触发事件表、连续低心态计数。

### 1. 金钱扣减口径:按余额比例而不是硬扣固定数

当前内容里的金钱扣减如果直接写死,会出现两个问题:一是扣减金额可能大于玩家余额,最后被引擎钳到 0,结果页却显示一个不真实的大额扣款;二是同一个事件对穷玩家和富玩家的压力完全不同,固定金额很难同时平衡两端体验。

后续金钱扣减统一按"当前余额百分比 + 上下限"设计,而不是直接写死负数。正向收入、工资、奖金、投资收益仍可用固定数,因为它们代表外部新增现金;负向支出、亏损、消费、诈骗、医疗、买房等都应进入比例扣减体系。

| 扣减类型 | 建议比例 | 上限/下限 | 设计用途 |
|---|---:|---|---|
| 日常小额支出 | 3%–8% | 上限 3000–8000 | 感冒、短途旅行、礼金、普通复查等,制造轻微压力但不清空人生 |
| 中等消费/预付费 | 10%–25% | 上限 20000–50000 | 健身课、副业课、长租公寓、装修、家庭支援 |
| 大额风险损失 | 30%–70% | 上限按事件设定 | P2P、基金追高、诈骗、杠杆爆仓,让富玩家也会疼 |
| all-in 决策 | 50%–100% | 按事件设置上下限 | 买房首付、重仓投资、重大医疗兜底等,应显著改变资产结构 |
| 资产置换 | 比例扣现金 + 后续资产回补 | 回补低于早期上车 | 买房这类事件先扣可用现金,后续通过新闻/估值事件把房产净值折回金钱 |

计算规则:

1. 先用 `当前余额 × 比例` 得到目标扣减。
2. 再套用事件配置的 `min/max`。
3. 实际扣减永远不能超过当前余额。
4. 结果页显示实际扣减,不是配置里的理论值。
5. 如果余额为 0,扣款为 0,但事件仍可通过心态、健康、人脉惩罚表达压力。

买房特殊规则:

- 买房不再固定扣 `-10000`、`-250000` 或 `-300000`,而是视为资产置换:先按比例扣可用现金,后续再折回部分房产净值。
- 2016 早买房扣当前余额 50%,后续房价新闻补回较高房产净值。
- 2021–2023 晚买房也扣当前余额 50%,后续补回房产净值,但金额明显低于早买房,表达"不是归零,但红利少很多"。
- 在没有拆分"现金 / 房产 / 负债"三栏之前,`money` 仍代表可用于结局判定的综合资产值,不是纯现金。

内容作者约束:

- 新事件里不要直接写大额 `stats: { money: -固定金额 }`;当前 `validate` 会拦截超过 10000 的固定负扣款。万元以内的小额叙事支出可以保留。
- 需要扣钱时优先使用比例扣减 DSL;需要买房/资产置换时使用"`moneyCost` 扣现金 + schedule 资产回补"结构。
- 所有大额扣减事件都要跑 `simulate --check`,看提前结局、兜底结局和金钱分位是否失衡。

## 四、职业线(5 条精做)

| 专业 | 时代主线 | 关键节点 |
|---|---|---|
| **计算机** | 互联网黄金十年全程 | 2018 进大厂 → 996/福报 → 2021 反垄断 → 2022 裁员潮 → 2023 AI(威胁或跳板) |
| **金融** | 资本市场过山车 | 2015 股灾实习震撼教育 → 2018 P2P 暴雷/资管新规 → 2020 基金热 → 2021 抱团崩塌 → **2022 降本增效(前台/中后台转档)** → 2023 AI 冲击投研 → 2024 限薪/出海 → 2025 天花板 |
| **师范** | 教育行业剧变 | 编制 vs 教培二选一 → 2021 双减(**强制转折**)→ 考编热 |
| **医学** | 慢热长线 | 5 年本科+3 年规培的煎熬(2019 规培中段)→ 2020 疫情主角线(全游戏最高光剧情)→ 编制与尊严 → **2022 挤兑洪峰(公立/民营转档)** → 2024 集采 → 2025 职称天花板 |
| **心理学** | 冷门到风口 | 2018 就业三岔口(学校/机构/转行)→ 2020 疫情心理热线 → 2021 双减青少年需求 → **2022 需求暴涨与耗竭(含转行支线)** → 2023 需求爆发/个人执业 → 2024 行业整顿 → 2025 助人者的自我照顾 |

> 注:计算机线的"软件工程"专业已并入"计算机科学与技术"(同一条职业线的重复入口,已删)。
>
> 注:5 条职业线专属事件已全年份覆盖到 2026(计算机 10 / 心理 9 / 医学 9 / 师范 8 / 金融 10)。其中医学 2022《这一年的白大褂》与金融 2022《降本增效那一年》设为"分支切换点":玩家可在公立↔民营、前台↔中后台等**既有收入档**之间转档(只改收入,不影响各线结局条件),是不同职业重玩辨识度的主要来源之一(M5 第二十八轮)。

大四统一三岔口:**考研 / 求职 / 考公**(考研成功推迟 3 年入场,改变后续所有事件年份)。

## 五、NPC 角色线(持续状态机,5 个固定角色)

| 角色 | 人设 | 跨年剧情弧 |
|---|---|---|
| 创业室友 | 折腾型 | 2015 双创拉你入伙(投不投?)→ 2018 失败 → 2020 直播翻身或消失 |
| 初恋 | 高中/大一相识 | 异地 → 毕业季抉择 → 结婚线或"多年后朋友圈点赞"线 |
| 卷王同学 | 镜像对照组 | 保研 → 大厂 → 也被裁,每个阶段与你互为参照 |
| 县城发小 | 留在家乡 | 考公上岸,每年春节饭局对照戏(城乡两种人生互相羡慕) |
| 职场贵人 | 导师型 | 人脉>阈值解锁,内推/点拨/临别赠言 |

每个 NPC 维护:好感度 + 剧情阶段指针,事件按阶段推进而非随机。

## 六、投资线(贯穿副线,金钱与心态的赌场)

2015 股灾(第一课)→ 2016 **买不买房**(全游戏影响最大的单一决策)→ 2017 比特币 → 2018 P2P 暴雷(高息诱惑陷阱)→ 2020 基金抱团 → 2021 白酒/中概崩 → 2024 黄金/出海。
设计原则:事后看全是明牌,身处其中全是迷雾——文案要还原当时的信息环境。

## 七、时间结构(变速粒度,约 15~16 回合)

| 阶段 | 粒度 | 回合数 |
|---|---|---|
| 2014.6 高考+志愿 | 事件级 | 2 |
| 2014–2018 大学 | 每学年 | 4(每回合:时代背景简报+2~3 事件) |
| 大四三岔口 | 特殊决策回合 | 1 |
| 2018–2026 社会 | 每年 | 8(背景简报+主线事件+随机事件池抽取) |

强制时代节点(不可跳过):2020 疫情、2021 双减(师范线)、2022 裁员潮(计算机线)、2023 ChatGPT 冲击。

## 八、结局系统(照搬 vc-simulator 引擎并改良)

- 判定函数:所有结局按 priority 升序,返回第一个 condition 为 true 的;兜底结局「普通人」。
- **提前结局**(每回合结算检查):心态≤0(休学/裸辞消失)、金钱破产、特殊事件链(如 P2P 全仓暴雷)。
- **2026 结算结局**(20~25 个),示例:
  - 「小镇做题家的胜利」农村家境 + 学识>80 + 金钱>百万
  - 「上岸人」考公成功 + 心态>60
  - 「北漂十年」一线城市 + 无房 + 金钱中等 + 心态>40
  - 「提前退休」比特币/早期投资 huge_success + 金钱>500万
  - 「双减幸存者」师范线转型成功
  - 「白衣执甲」医学线 + 疫情事件高光选择
  - 「30 岁,重新开始」被裁 + 心态>50(温情向)
  - 「围城内外」结婚线/单身线各有专属结局
- 改良:不做 vc-simulator 那种 `() => false` 死代码结局;每个结局配分享卡片(称号+数值雷达图+一句话总结)。

## 九、技术方案

- React 18 + Vite + TypeScript + Zustand(与 vc-simulator 同栈,其模式已验证)
- 带种子的 RNG(mulberry32),同一种子可复现整局 → 方便测试与"挑战同一人生"分享玩法
- 事件用纯数据定义:`{ id, 年份/阶段, 触发条件(状态)=>bool, 文案, choices: [{text, statChanges, 后续标记}] }`
- localStorage 存档(vc-simulator 没做,我们要做)
- 静态部署(Vercel/Cloudflare Pages)

## 十、首版范围(MVP)

1. 完整流程:抽卡→高考→志愿→大学 4 年→三岔口→社会 8 年→结算
